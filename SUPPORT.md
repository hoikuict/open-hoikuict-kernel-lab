# サポート

## サポート範囲

このプロジェクトは公開準備・検証段階のオープンソースソフトウェアです。商用SaaSのような稼働保証、個別園への導入保証、障害対応SLAはありません。

## 問い合わせ先

- 一般的な質問・改善提案: GitHub Issues
- セキュリティに関する連絡: `openhoikuict@gmail.com`
- 公式サイト: <https://open.hoikuict.net>

## 実運用で必ず確認すること

- 園・法人としての運用責任者
- 個人情報保護方針と保護者への説明
- 認証、権限、退職者アカウント停止
- バックアップと復旧手順
- 障害時の紙運用・代替連絡手段
- 自治体・監査・補助金要件への適合
