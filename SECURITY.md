# セキュリティポリシー

## 対象バージョン

現時点では正式リリース前のため、サポート対象バージョンは定義していません。公開デモ・開発版では実在する個人情報を扱わないでください。

## 脆弱性の報告

セキュリティ上の問題を見つけた場合は、公開Issueではなく `openhoikuict@gmail.com` へ連絡してください。

報告に含めてほしい情報:

- 影響を受ける画面・機能
- 再現手順
- 想定される影響
- 可能であれば修正案

## 禁止事項

- 実在する園児・保護者・職員の情報を含む検証
- 第三者の環境に対する無断スキャン
- 公開デモに対する高負荷試験
- 認証回避やデータ取得を目的とした攻撃的検証

## 本番利用前の最低条件

- 本番認証とパスワード再設定
- ロールごとの権限制御
- 保護者が他家庭の情報を見られないことの検証
- 主要操作の監査ログ
- TLS、バックアップ、復旧手順
- 退職・異動時のアカウント停止手順
