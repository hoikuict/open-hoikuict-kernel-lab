# 個人情報・ダミーデータ方針

## 個人情報を入れない開発

開発、Issue、Pull Request、スクリーンショット、デモ環境では、実在する園児・保護者・職員の情報を扱わないでください。

避けるべき情報:

- 氏名、住所、電話番号、メールアドレス
- 生年月日、健康情報、アレルギー、診断名
- 家庭状況、勤務先、緊急連絡先
- 登降園時刻、欠席理由、連絡帳の実内容
- 顔写真、園を特定できる画像

## このバンドルのダミーデータ

同梱データはすべて架空です。

- メールアドレス: `demo.open-hoikuict.example`
- 住所: 東京都サンプル区などの架空表記
- 電話番号: `03-0000-xxxx`, `090-000x-xxxx` 形式の架空番号
- 園児・保護者名: デモ用に生成した組み合わせ
- 医療・アレルギー情報: デモ機能確認用の架空内容

## 実運用時に決めること

- 利用目的
- 保護者への説明文
- 保存期間
- 訂正・削除依頼への対応
- 職員の閲覧範囲
- 外部委託先・サーバー管理者の範囲
- 漏えい疑い時の報告手順
