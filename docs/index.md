# open-hoikuict

## 現場で確かめながら育てる、保育園のためのICT

open-hoikuict は、園児名簿、家庭・保護者管理、登降園、保護者からの日次連絡、出欠確認、園内連絡を中心に、保育現場の運用を少しずつ整理していくためのオープンなプロジェクトです。

!!! warning "現在の位置づけ"
    このプロジェクトは公開準備・デモ・検証段階です。実在する園児・保護者・職員の個人情報を投入しないでください。本番導入には、認証、権限、監査ログ、バックアップ、運用責任者、サポート体制の整備が必要です。

## 誰向けか

- 保育園職員、園長、主任、事務担当者
- 小規模なICT化を自分たちで進めたい園
- GAS やスプレッドシート運用を、より保守しやすい形に整理したい人
- 保育ICTの業務フローを試作・検証したい開発者

## 何ができるか

- 園児名簿と家庭単位の情報確認
- 登降園時刻、迎え予定者、出欠確認
- 保護者からの欠席・体調・日次連絡
- 連絡内容と目視確認の不整合アラート
- お知らせ、既読確認、職員ルーム、カレンダー
- 健康情報、アレルギー、健診記録

## このプロジェクトについて

完成品を一方的に配布する場所ではなく、現場で必要な運用と実装を一緒に育てていく場所です。便利さだけでなく、権限漏れの防止、個人情報保護、引き継ぎ、監査可能性、障害時の代替手段を重視します。
